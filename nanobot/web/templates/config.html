Send_Progress

Khi bật, bot gửi tin nhắn trung gian trong lúc xử lý (ví dụ: "Đang tìm kiếm...", "Đang đọc file..."). Giúp user biết bot đang làm gì thay vì im lặng chờ. Nên tắt nếu channel không hỗ trợ edit message hoặc gây nhiễu.

Send_Tool_Hints

Khi bật, bot thông báo ngắn khi bắt đầu dùng tool (ví dụ: "🔍 Đang tìm kiếm web...", "📂 Đang đọc file..."). Giúp user hiểu bot đang thực hiện thao tác gì. Khác Send_Progress: chỉ thông báo tên tool, không phải toàn bộ intermediate output.
Debounce

Debounce

Thời gian chờ (giây) trước khi xử lý tin nhắn. Nếu user gửi nhiều tin liên tiếp trong khoảng này, bot gộp tất cả thành 1 lần xử lý.

Ví dụ (debounce = 2s):
• "Bạn ơi..." → chờ 2s
• "cho tôi hỏi..." → reset timer
• "về Python nhé" → gộp 3 tin, trả lời 1 lần Đặt 0 để tắt. Khuyến nghị: 1.5–3s.
